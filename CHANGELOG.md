# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-06-19

### Added

- Initial Blender addon UI for RizomUV round-trip workflow
- Single-object send and fetch operations
- External helper architecture using RizomUV's bundled `python.exe`
- Batch workflow for sending multiple selected mesh objects together
- Batch UV fetch back into Blender
- UV copy from active object to selected meshes with matching topology
- Blender-side and helper-side logging for debugging
- Public project documentation and repository metadata
